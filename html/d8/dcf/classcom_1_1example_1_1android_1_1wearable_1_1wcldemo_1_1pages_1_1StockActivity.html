Detailed Description

A simple activity that makes an HTTP request to get the GOOGLE stock price and if successful, presents the result to the user. The HTTP request is made using the WearHttpHelper helper class.

Called, on the main thread, when the response to the HTTP request is available.

Parameters
requestIdA unique id that was created and associated with this request
statusThe status code of the response. Positive numbers reflect the HTTP status and negative values are custom errors, defined in the WearHttpHelper class.
responseThe text-based response of the request. If there is an error, this can be null
