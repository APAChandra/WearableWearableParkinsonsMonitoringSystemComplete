Detailed Description

An implementation of NodeSelectionFilter which allows narrowing down a list of nodes to at most one node. Clients can use this filter through a pattern such as: 

    new SingleNodeFilter(aNodeSelectionFilter).filterNodes(nodes)

If "aNodeSelectionFilter" is non-null, then the above pattern first applies aNodeSelectionFilter to the set of

nodes

and then a single node from the result is picked and returned. if "aNodeSelectionFilter" is

null

, a single node from the original set of

nodes

will be selected.
