Detailed Description

A fragment that shows how a wear application can use the companion app on a phone to perform network calls on its behalf. In this fragment, when we receive such a request from the wear device, we show a bit of information about the request, such as the url for the request and the nodeId of the node that sent the request. Then this fragment makes the network call and when it has the response, it forwards the response to the originating node and show the status of the response on the screen.
