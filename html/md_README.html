WearableParkinsonsMonitoringSystem

Movement monitoring using Moto360 watch and other tests for parkinsons. Here is the full documentation.

Overview

Wear App

The app for the wearable device 'Moto 360' which is used to record the sensor values when instructed by the mobile phone and store it in the database.




The database can be pushed to the phone.


Select File Transfer option and then click on Send DB to transfer the DB file to the android phone.


Select Launch on Mobile option to open the app on the Android phone.

Mobile App

The app for the Android mobile phone which has the following functionalities: a) Register new user.


b) Register Medical History of the patient.



c) Record Sensor values for Rest Tremor Tests. To start the Android Watch sensor recording, select the Play button the phone.



d) Perform Cognitive test.


e) Perform Bradykinesia test.



f) To get the database file from the Android watch select File Transfer option.


Flexibility

Both of our apps are independent and can be used individually as they also have different database files.

Installation

  1. git clone https://github.com/deshmukhrajvardhan/WearableParkinsonsMonitoringSystem.git
  2. Install Android studio
  3. Open Android studio: choose File->Open File or Project-> <path> WclDemoSample and WearCompanionLibrary together.
  4. Android App: Enable developer options on your phone.

For Android Wear

  1. Install the Android Wear application on your phone from the app store.
  2. Connect the watch to phone via bluetooth.
  3. Connect the watch to the phone.
  4. Enable the developer options on the Android watch.
