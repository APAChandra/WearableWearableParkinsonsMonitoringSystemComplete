Detailed Description

A WearableListenerService that is started when the client application is in front. WearManager uses this as a means to get a number of wear related callbacks and as such, WearManager starts this service by calling

Context.startService()

which then makes this service a long-lived service. WearManager is also responsible for stopping this service; clients could call WearManager#stopWearableService() to do that or the WearManager will do so when the client application is no longer visible.
